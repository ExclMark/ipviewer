# IpViewer
* A tool to get information about any IP
# Requirements:
* Python 3.x
* requests
# Installation:
* ```git clone https://github.com/ExclMark/ipviewer```
* ```pip3 install -r requirements.txt```
# Usage:
* ```cd ipviewer```
* ```python3 ipviewer.py```
